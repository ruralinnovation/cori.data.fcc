# Package index

## NBM

- [`get_nbm_bl()`](https://ruralinnovation.github.io/cori.data.fcc/reference/get_nbm_bl.md)
  : Load NBM service counts for given Census Block from CORI S3 bucket
- [`get_county_nbm_raw()`](https://ruralinnovation.github.io/cori.data.fcc/reference/get_county_nbm_raw.md)
  : Load all BSL from NBM dataset for a specific county
- [`get_frn_nbm_bl()`](https://ruralinnovation.github.io/cori.data.fcc/reference/get_frn_nbm_bl.md)
  : Load part of NBM at Census Block from CORI s3 bucket
- [`get_nbm_release()`](https://ruralinnovation.github.io/cori.data.fcc/reference/get_nbm_release.md)
  : Get a list of release available in FCC NBM
- [`get_nbm_available()`](https://ruralinnovation.github.io/cori.data.fcc/reference/get_nbm_available.md)
  : Get a list of files availables in FCC servers
- [`dl_nbm()`](https://ruralinnovation.github.io/cori.data.fcc/reference/dl_nbm.md)
  : Download NBM data

## Form 477

- [`get_f477()`](https://ruralinnovation.github.io/cori.data.fcc/reference/get_f477.md)
  : Load part of Form 477 estimates data from CORI s3 bucket

## Environment Variables & Utilities

- [`get_fcc_dictionary()`](https://ruralinnovation.github.io/cori.data.fcc/reference/get_fcc_dictionary.md)

  : Get info on one or nor FRN
- [`fcc_provider`](https://ruralinnovation.github.io/cori.data.fcc/reference/fcc_provider.md)
