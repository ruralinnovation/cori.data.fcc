# FCC providers list

FCC list of preoviders downloaded in May 2024

## Usage

``` r
fcc_provider
```

## Format

### `fcc_provider`

A data frame with 4,341 rows and 5 columns:

- provider_name:

  Provider name

- affiliation:

  Affiliation

- operation_type:

  ILEC or not

- frn:

  FCC Registration Number, number of the entity that submited the data

- provider_id:

  Provider ID
